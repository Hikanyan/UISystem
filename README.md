# UISystem

A simple UI system for Unity.

## Installation

Add this package to your Unity project via Package Manager:

1. Open Unity Package Manager (Window > Package Manager).
2. Click the "+" button and select "Add package from git URL".
3. Enter the GitHub URL: `https://github.com/your-username/UISystem.git`

Or add to your `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.koiirohikanayn.uisystem": "https://github.com/your-username/UISystem.git"
  }
}
```
